# Nexalume Solutions

Premium AI-first technology company website built with Next.js, TypeScript, Tailwind CSS, Framer Motion, and GSAP.

**Live domain:** [nexalume.in](https://nexalume.in)

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion, GSAP
- **UI:** Shadcn-style components (Radix UI)
- **Icons:** Lucide React

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
npm start
```

## Project Structure

```
src/
├── app/              # Next.js App Router pages & layout
├── components/
│   ├── animations/   # AI background, scroll reveal, parallax
│   ├── effects/        # Mouse glow, scroll progress
│   ├── layout/         # Navbar, Footer, floating buttons
│   ├── sections/       # All page sections
│   └── ui/             # Reusable UI components
└── lib/                # Data & utilities
```
